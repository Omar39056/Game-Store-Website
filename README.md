 Gaming  Website

A responsive and interactive multi-page website designed for gaming enthusiasts, featuring modern UI/UX elements.

## Features
- **Responsive Design:** Fully compatible with desktops, tablets, and mobile devices.
- **Dynamic Content:** Interactive elements powered by JavaScript to enhance user engagement.
- **Modern UI:** Styled with CSS3 including transitions, hover effects, and a dark-themed gaming aesthetic.
- **Multi-page Navigation:** Includes sections for game categories, news, and community features.

##  Tech Stack
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Advanced styling, Flexbox, and Grid for layout management.
- **JavaScript (ES6):** Logic for interactive components and UI manipulation.
